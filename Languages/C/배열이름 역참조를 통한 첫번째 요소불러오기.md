- C언어에서 배열은 포인터이다. 그리고 배열의 이름은 배열의 주소를 가르키며, 이는 배열의 첫번째 요소를 가르킨다.

- 이러한 성질을 이용하여 배열 이름에 대한 역참조를 통해서 배열의 첫번째 값을 불러올 수 있다.

```C
#include <stdio.h>

int main()
{
    int numArr[10] = {11,22,33,44,55,66,77,88,99,1010};

    printf("%d\n",*numArr); 
    printf("%d\n",numArr[1]);
    printf("%d\n",numArr[7]);
}
/*
Reuslt
11
22
88
*/
```
